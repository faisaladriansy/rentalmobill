🎬 Skenario Use Case Detail
📋 Kelola Data Transaksi
Aktor: Admin Rental
Deskripsi: Admin mengelola semua data transaksi rental mobil
Langkah: 1. Admin login ke sistem → 2. Pilih menu "Data Transaksi" → 3. Lihat daftar transaksi → 4. Tambah/Edit/Hapus transaksi → 5. Update status transaksi
🚗 Kelola Data Mobil
Aktor: Admin Rental
Deskripsi: Admin mengelola data mobil dan status ketersediaan
Langkah: 1. Admin masuk ke menu "Data Mobil" → 2. Lihat daftar mobil dengan status → 3. Tambah mobil baru dengan pilih merk → 4. Edit data mobil → 5. Update status (Tersedia/Jalan)
➕ Tambah Transaksi Baru
Aktor: Admin Rental
Deskripsi: Admin membuat transaksi rental baru untuk customer
Langkah: 1. Admin klik "Tambah Transaksi" → 2. Input data customer (nama, KTP, HP, alamat) → 3. Pilih mobil yang tersedia → 4. Pilih supir atau lepas kunci → 5. Set tanggal sewa dan kembali → 6. Sistem hitung otomatis biaya → 7. Simpan transaksi → 8. Status mobil otomatis berubah "Jalan"
✅ Selesaikan Transaksi
Aktor: Admin Rental
Deskripsi: Admin menyelesaikan transaksi saat mobil dikembalikan
Langkah: 1. Admin cari transaksi aktif → 2. Klik button "Selesai" → 3. Sistem update tanggal kembali → 4. Sistem hitung total biaya final → 5. Status transaksi berubah "Selesai" → 6. Status mobil otomatis berubah "Tersedia" → 7. Generate invoice PDF
🔄 Update Status Mobil
Aktor: Admin Rental
Deskripsi: Admin mengubah status mobil secara manual
Langkah: 1. Admin buka "Data Mobil" → 2. Lihat status mobil saat ini → 3. Klik button status (Jalan/Tersedia) → 4. Konfirmasi perubahan → 5. Sistem update status di database → 6. Status terupdate di semua tampilan
