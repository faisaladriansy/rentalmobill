🔍 Penjelasan Error dan Perbaikan

1. Missing @startuml/@enduml Tags

Error: Tidak ada tag pembuka dan penutup

Fix: Tambahkan @startuml di awal dan @enduml di akhir

2. Missing Spaces

Error: useCaseDiagramactor - tidak ada spasi

Fix: actor "Customer" as C - dengan spasi yang proper

3. Wrong Relationship Syntax

Error: C --( UC1 - syntax salah

Fix: C --> UC1 - menggunakan panah yang benar

4. Include/Extend Relationships

Correct syntax:

  • UC1 .> UC4 : <> - untuk include
  • UC_Process .> UC1 : <> - untuk extend

5. Proper Formatting

Gunakan line breaks dan indentasi yang proper untuk readability

✅ Cara Menggunakan:

  1. Copy kode yang sudah diperbaiki dari tab di atas
  2. Paste ke PlantUML editor seperti:
    • PlantUML Online
    • PlantUML Editor
  3. Generate diagram untuk melihat hasilnya
  4. Export dalam format PNG, SVG, atau PDF

💡 Tips PlantUML:

  • Actors: actor "Name" as Alias
  • Use Cases: usecase "Name" as Alias
  • Associations: Actor --> UseCase
  • Include: UseCase1 .> UseCase2 : <>
  • Extend: UseCase1 .> UseCase2 : <>
  • Comments: ' This is a comment
